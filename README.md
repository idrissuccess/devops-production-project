#  DevOps Docker AWS Project

## 📌 Overview
Deployed a containerized web application on AWS EC2 using Docker and Nginx.

## 🛠 Tools & Technologies
- AWS EC2
- Docker
- Nginx
- Linux (Ubuntu)

## ⚙️ What I Did
- Created a simple web application
- Containerized the application using Docker
- Deployed the container on AWS EC2
- Exposed the app on port 80 for public access

## 🌐 Architecture
User → AWS EC2 → Docker Container → Nginx → Application
